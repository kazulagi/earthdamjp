# earthdam.jp

``` earthdam.jp ```は、日本の農業用ため池のデータベースです。
日本のため池について、

- 来歴
- 土質データ
- メッシュデータ

をとりまとめて公表しています。

全てのデータは、MITライセンスの下で無料でご利用頂けます。
